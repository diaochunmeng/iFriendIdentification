友贱安卓版 究竟哪个jian人偷偷删你的微信?

使用方式
* 微信扫码获得身份认证
* 程序会创建空的群聊，逐一邀请好友到群聊，根据微信的返回码确认ta是否偷偷删了你
* 随后会把好友请出聊天室，全程所有好友无感知


关于本程序：
* 依据0x05提供的[开源程序](https://github.com/0x5e/wechat-deleted-friends)移植到安卓平台
* 不会记录上传任何微信用户数据! 引入了友盟想看看用户安装量，不需要可以去掉。
